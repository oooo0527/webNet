# 移动端明星网站

这是一个基于Vue.js的移动端适配网站，包含四个主要模块：明星档案、数据工坊、作品安利和粉丝心声。

## 功能特点

1. **响应式设计**：完全适配移动设备屏幕
2. **用户系统**：支持游客访问和注册登录访问
3. **四大模块**：
   - 明星档案：查看明星详细资料
   - 数据工坊：数据分析与可视化
   - 作品安利：推荐优秀作品
   - 粉丝心声：粉丝互动交流区

## 技术栈

- Vue.js 3
- Vue Router 4
- Vuex 4
- ES6+

## 项目结构

```
src/
├── assets/           # 静态资源
├── components/       # 公共组件
├── router/           # 路由配置
├── store/            # 状态管理
├── utils/            # 工具函数
├── views/            # 页面组件
└── App.vue           # 根组件
```

## 安装与运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run serve

# 构建生产环境
npm run build
```

## 移动端适配特性

1. **响应式布局**：使用CSS Grid和Flexbox实现响应式布局
2. **触摸优化**：按钮和交互元素符合移动端触摸标准
3. **视口设置**：正确配置viewport meta标签
4. **安全区域适配**：支持iPhone X等异形屏设备
